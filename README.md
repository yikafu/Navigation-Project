# Navigation-Project

基于vite的导航页面模板

**环境要求**

- node >= 18
- vite 4.5.0
- yarn 1.22.19

**拉取代码并安装依赖**

```
git clone https://github.com/yikafu/Navigation-Project.git
cd ./Navigation-Project
yarn init
```

**修改配置文件**

修改 src/data/data.json文件中的内容。

**运行**

```
yarn dev
```

**打包编译**

```
yarn build
```
